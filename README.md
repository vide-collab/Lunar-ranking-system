# Lunar Ranking System

Creator tier ranking system for X (Twitter) influencer campaigns.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features

- Dual gate scoring: Smart Followers + Median Impressions
- Median Impressions weighted at 35% for CPM optimisation
- 5 tiers: Inner Circle, Advanced, Established, Emerging, Beginner
- Performance score breakdown per creator
- Prior payout vs new rate comparison
- Estimated CPM calculation
- CSV export of full roster
- Search by handle or display name

## Tech Stack

- Next.js 14
- React 18
